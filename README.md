# Real-Time Power BI Project: Uber Analysis

## Project Overview
This project is a real-time data analysis of Uber trips using Power BI. It aims to provide insights into trip patterns, locations, and other key metrics.

## Contents
- **ok.pbix**: The main Power BI dashboard file. Open this file to view the analysis and visualizations.
- **Uber Trip Details.xlsx**: Source data containing detailed information about Uber trips.
- **Location Table.xlsx**: Source data containing location-specific details.
- **Problem Statement.docx**: Document outlining the problem statement and objectives of the analysis.
- **Real Time Power BI Project.docx**: Documentation regarding the project.
- **Images/** & **deshbord_images/**: Folders containing related images and dashboard screenshots.
- **Uber Data...zip**: Compressed source data files.

## Getting Started
1. **Prerequisites**: Ensure you have [Microsoft Power BI Desktop](https://powerbi.microsoft.com/desktop/) installed.
2. **Open the Report**: Double-click `ok.pbix` to open the project in Power BI.
3. **Data Refresh**: If you wish to refresh the data, ensure the Excel files (`Location Table.xlsx` and `User Trip Details.xlsx`) are in the correct path or update the data source settings in Power BI to point to their current location.

## Analysis Highlights
- Comprehensive analysis of trip data.
- Visualization of pickup and drop-off locations.
- Insights into trip duration, frequency, and trends over time.
